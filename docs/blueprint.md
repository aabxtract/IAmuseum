# **App Name**: Internet Artifact Museum

## Core Features:

- Artifact Upload & Minting: Users upload digital artifacts with metadata, generating an archive hash (keccak256) and minting the artifact as an ERC721 NFT.
- Archive Gallery: Display minted artifacts in a museum-style gallery with thumbnails, titles, eras, category badges, and uploader wallet information.
- Exhibit Page: Show a large artifact preview with details like name, era, description, archive hash, mint date, contributor address, and category tags, plus social sharing buttons.
- DAO Voting System: Implement a DAO voting system where members vote on Artifact of the Week, featured eras, and artifacts for the Main Hall, using a governance token. Includes live vote results display.
- Main Hall Curation: Curate a featured section (Main Hall) of top-voted artifacts, with animated spotlight and glowing frame effects to enhance visual appeal.
- Profile Page: User profile page to see their contributions, votes and DAO participation metrics

## Style Guidelines:

- Primary color: Retro Blue (#7DF9FF) to capture the essence of early internet aesthetics.
- Background color: Soft Beige (#F5F5DC) for a nostalgic and archival feel.
- Accent color: Old Terminal Green (#00FF41) to mimic vintage computer terminals and highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, machined look, suitable for headlines or body text
- Pixel-style icons and graphics to reinforce the retro internet aesthetic.
- Use a grid-based layout with soft grid lines to create a museum-like gallery for displaying artifacts.
- Incorporate smooth hover animations and CRT monitor glow effect to enhance the user experience and create a nostalgic atmosphere.
